## Live Projects

### 1. AI Quality Audit Evaluator
**[→ View Live Tool](https://guyhlaroche-eng.github.io/laroche-lab-portfolio/ai_quality_audit_evaluator.html)**

A structured quality evaluation framework for AI-generated customer support responses.

Scores outputs across 4 dimensions — Accuracy, Tone, Policy Compliance, and Hallucination Risk — using a rubric-based methodology derived from operational auditing practice at Amazon scale.

**What it includes:**
- 4-dimension scoring rubric with defined criteria for every score level (1–5)
- 10 fully scored test cases spanning simple queries to complex multi-issue interactions
- Flag thresholds: Pass / Pass with note / Review / Fail / Critical Fail
- 3 identified systemic failure patterns with actionable remediation guidance
- Per-case key findings structured for engineering and product team consumption

**Core finding:** 7 of 10 evaluated responses exhibited the *Confident Fabricator Pattern* — the AI invents specific details (dates, amounts, reference numbers, feature names) when it lacks grounded system data, rather than acknowledging uncertainty.

**Built with:** Vanilla HTML/CSS/JS — no dependencies, no frameworks, fully portable.
